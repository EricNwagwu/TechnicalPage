HTML

HTML is the standard markup language for creating Web pages. HTML stands for Hyper Text Markup Language. HTML describes the structure of a Web page.

List of HTML Tags:

  • header - contains things that would be in the title of your page
  • body - contains everything that the user sees goes here
  • main - contains the main portion of your web page
  • div - general purpose tag
  • footer - contains stuff at bottom of page such as copyright and publishing date
  • a - creates a hyper link that goes to the href link
  • img - this tage is self closing and displays image from src link
  • p - this tag is used for paragraphs
  • ul - creates an unordered list
  • ol - creates an ordered list
  • li - this is a list element used in the ul and ol tags
  • form - this element takes in information from the user and send it to the server
  • button - creates a clickable element that can be used to complete actions
  • input - this tag is self closing and creates various input types base on its type property
  • label - this element is used to add text to your input elements
  • br - this element is self closing and goes to the next line in text
  • head - contains the metadata elements such as links, meta, title and style
CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Example of CSS code:

h1{ color: red; }//this makes the h1 element's text color red

List of CSS Properties:

  • color - color: "color name";//changes color of text
  • font-size - font-size: "size";//changes the font size of text
  • font-family - font-family: "font name";//changes the font
  • width - width: "size";//changes the width of an element
  • border-color - border-color: "color name";//changes the border color of elements
  • border-width - border-width: "size";//changes the thickness of border
  • border-style - border-style: "type of border ex..solid";//changes the type of border
  • border-radius - border-radius: "size";//creates rounded edges
  • background: "color name";//changes element color
  • padding - padding: "padding top size" "padding right top" "padding bottom size" "padding left size";// changes amount of space between border and content
  • margin - margin: "margin top size" "margin right top" "margin bottom size" "margin left size";//changes distance between border and other elements

CSS Flexboxes

CSS3 introduced Flexible Boxes, or flexbox, to create page layouts for a dynamic UI. It is a layout mode that arranges elements in a predictable way for different screen sizes and browsers. While somewhat new, all popular modern browsers support flexbox

Example of CSS code using flexbox:

display: flex;//sets display to flex
CSS Grid

In the CSS grid section of freecodecamp, I learned that the CSS Grid helps you easily build complex web designs. It works by turning an HTML element into a grid container with rows and columns for you to place children elements where you want within the grid.

Example of CSS code using grid:

display: grid;//set display to grid
